## Quick Reference

### Goals Summary
- Automate demand letter generation to reduce drafting time by 50%
- Enable firm-specific template management and customization
- Provide AI-powered refinement and editing capabilities
- Increase attorney productivity and client satisfaction

### Primary Users
- Attorneys (primary)
- Paralegals and Legal Assistants (secondary)

### Key Success Metrics
- 50% reduction in demand letter drafting time
- 80% user adoption rate within first year
- Increased client retention and satisfaction scores
- Generation of new sales leads through innovative AI capabilities

### Technical Stack
- Frontend: React
- Backend: NodeJS, Python
- Infrastructure: AWS Lambda, PostgreSQL
- AI: Anthropic API or AWS Bedrock
